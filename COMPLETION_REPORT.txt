## 🐛 COMMON ISSUES & SOLUTIONS

### Issue: "This site can't be reached"
   Solution:
   1. Make sure server window is open
   2. Check "Server running on port 3000" message
   3. Try http://127.0.0.1:3000
   4. Restart server using START_PRO_SERVER.bat

### Issue: Port already in use
   Solution:
   - The batch file handles this automatically
   - Or manually kill the process:
     netstat -ano | findstr :3000
     taskkill /F /PID <PID>

### Issue: Login doesn't work
   Solution:
   1. Use demo credentials: admin@bank.com / password123
   2. Create a new account
   3. Check password is correct
   4. Verify server is running
